# nellY_DETECT

A real-time AI-powered object detection and tracking web application built using:

- Streamlit
- YOLOv8
- OpenCV
- Streamlit WebRTC

This project detects and tracks objects directly from a live webcam feed using the YOLOv8 model with ByteTrack tracking support.

---

# 🚀 Features

✅ Real-time object detection  
✅ Live webcam streaming  
✅ Object tracking with unique IDs  
✅ FPS (Frames Per Second) counter  
✅ Adjustable confidence threshold  
✅ Toggle bounding boxes and labels  
✅ Interactive Streamlit UI  
✅ Detects multiple object classes  

---

# 🛠 Technologies Used

| Technology | Purpose |
|---|---|
| Streamlit | Web Application Framework |
| YOLOv8 | Object Detection Model |
| OpenCV | Image Processing |
| Streamlit-WebRTC | Live Webcam Streaming |
| ByteTrack | Object Tracking |
| PyTorch | Deep Learning Backend |
